------------------------------------------------------------
Common failure points
------------------------------------------------------------
1) Missing include for InputMode/InputState/InputContext.
2) Missing extern for resetInputContext or data globals.
3) Header signature mismatch vs implementation.
4) Default arguments duplicated in both .h and .cpp (default only in .h).
